# Descrição da Feature

Feature criada com a finalidade e objetivo de desenvolver a implementação de sessão de newsletter. Que funcionará da seguinte maneira:

- Irá notificar os assinantes sempre que um novo repositório for criado;
- Irá notificar os assinantes sempre que um novo video no canal do youtube for lançado;
- Irá notificar os assinantes sempre que um novo artigo no blog for postado.
