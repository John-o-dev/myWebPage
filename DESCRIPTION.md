# Descrição de Feature

Feature criada com a finalidade de desenvolver a área de galeria de certificados e de relacionar e mostrar projetos desenvolvidos no curso em questão.

#### Por exemplo:

- Cada certificado vai linkar para o projeto desenvolvido, se houver;
- Competências, habilidades desenvolvidas/aprendidas no curso;
- Mostrar tecnologia, framework, e/ou biblioteca utilizada;
- Descrição do curso.

__É uma forma de validar o que foi aprendido, dando mais credibilidade e contexto prático para o portfólio__
